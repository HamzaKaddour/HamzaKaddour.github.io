Wire and Transmission Lines Detection using YOLO

Computer vision project for detecting power lines and transmission towers in aerial imagery using YOLOv5, YOLOv8, and YOLOv11, evaluated on the TTPLA dataset.

PythonComputer VisionYOLODeep LearningObject DetectionUAVModel Benchmarking
GitHub Repo
Project overview

Focus

This project evaluates modern YOLO-based object detection models for power grid inspection using the TTPLA aerial-image dataset. The workflow covers data preparation, label handling, model training, and evaluation for challenging aerial scenes with complex backgrounds, thin wire structures, and class imbalance. The study compares YOLOv5, YOLOv8, and YOLOv11, with YOLOv8 achieving the strongest overall performance in the reported experiments.

Contributions

  • Evaluated YOLOv5, YOLOv8, and YOLOv11 for aerial power-line detection
  • Used the TTPLA dataset, which contains high-resolution aerial images with annotated bounding boxes
  • Addressed challenges such as complex backgrounds, thin wire structures, and class imbalance
  • Reported that YOLOv8 achieved the best overall precision/recall performance among the tested YOLO variants
